== Description ==

This is keyword rank checker in Classic Editor.You could view top rank contents rapidly.

This tool displays the title, the number of characters, and the headline of the content above the search order of the specified keyword, and helps you create the content.

Understanding what kind of headline articles appear at the top helps you with ideas for similar content and original content.

It is hard to open various tools while opening WordPress, but if this plug-in is inserted, everything will be completed in the editor.

Add-ons allow you to see in more detail the features and top-view content you can add headings with a single click.

This plugin:

* You can search top rank contents in each keyword.
* Get information from Google "Custom Search JSON API" and display it.
* If you want to know more about "Custom Search JSON API", look at [this site](https://developers.google.com/custom-search/v1/overview).
* You have to comply with ["Custom Search JSON API" terms of use](https://developers.google.com/custom-search/terms).

1. Upload the `top-rank-content-checker` directory to your `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
